# Royal Spice Restaurant Website

## Project Description
Royal Spice is a **premium luxury restaurant website** built with **HTML, CSS, and JavaScript**. It delivers a **responsive, modern** handcrafted UI/UX with a cinematic, warm dining theme—featuring elegant typography, glassmorphism-inspired components, and smooth interactions across devices.

## Features
- **Fully responsive design** (mobile, tablet, desktop)
- **Premium hero section** with cinematic styling
- **Sticky navbar** with mobile-friendly navigation
- **Food menu** with images and **prices in ₹ INR**
- **Gallery section** with dynamic image layout
- **Reservation form** for table booking requests
- **Customer testimonials** section
- **Smooth animations** (scroll reveal + subtle parallax)
- **Mobile-friendly layout** with optimized spacing and grid behavior

## Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript**
- **Flexbox**
- **CSS Grid**
- **Google Fonts**

- **Warm white and cream background**
- **Gold/orange accents**

## Responsive Design
The layout is designed to support:
- **Mobile**
- **Tablet**
- **Desktop**

Key sections (menu grid, testimonials, reservations, navbar) automatically adapt using media queries, Flexbox, and CSS Grid.
